## What is TeleVantage?

TeleVantage is a software-based PBX and phone system developed by Vertical Communications (originally Artisoft) that runs on Microsoft Windows operating systems [citation:3][citation:4]. It integrates your telephone and computer to make your telephone a more powerful tool, allowing you to manage calls and voicemail using either your telephone or the TeleVantage ViewPoint application on your computer [citation:1].

Infrastructure teams managing business communications benefit from TeleVantage's software-based architecture, which provides more options at a lower cost than traditional PBX hardware [citation:3]. System administrators appreciate the Windows-based interface, scalability up to 96 trunks and 264 extensions, and integration with contact management programs [citation:3].

## Key Features

| Feature | Description |
|---------|-------------|
| **Software-Based PBX** | Runs on Windows NT/2000/Server platforms, eliminating the need for proprietary PBX hardware [citation:3] |
| **ViewPoint Application** | Desktop client providing complete call and message control from any PC [citation:1][citation:3] |
| **Voice-Guided Menus** | Access most TeleVantage features using telephone keypad commands [citation:1] |
| **ViewPoint Web Access** | Browser-based access for remote users to check voicemail and manage personal settings [citation:3] |
| **Call Recording** | Record calls and save as .wav files; forward recordings to other users with electronic notes [citation:3] |
| **Follow-Me Call Routing** | Route calls to multiple locations (home, cell phone, office) with caller-specific forwarding rules [citation:3] |
| **Call Center Features** | Automatic Call Distribution (ACD) and call center reporting with Microsoft Excel integration [citation:3] |
| **IP Gateway** | Built-in IP gateway for routing calls over the Internet or intranet [citation:3] |
| **Contact Management Integration** | Works with Microsoft Outlook and Goldmine for point-and-click dialing and screen-pops on incoming calls [citation:3] |
| **Scalability** | Scales up to 96 trunks and 264 extensions [citation:3] |

## Installation Guide

### Prerequisites

- Windows Server or Windows 10/11 (depending on version)
- Intel Dialogic telephony components (for analog/T1/E1 connectivity) [citation:7]
- Network connectivity

### Step 1: Install Intel Telephony Components

**Step 1:** Determine the Intel telephony resources required for your TeleVantage system [citation:7].

**Step 2:** Install Intel HMP (Host Media Processing) software or Intel Dialogic boards and drivers [citation:7].

**Step 3:** Configure the telephony hardware.

### Step 2: Install TeleVantage Server

**Step 1:** Follow the Installing TeleVantage manual for server installation [citation:8].

**Step 2:** Configure the TeleVantage database server [citation:7].

**Step 3:** Configure trunks, stations, and users [citation:8].

### Step 3: Install ViewPoint Client

**Step 1:** On each workstation, navigate to **Start > All Programs > Vertical TeleVantage > TeleVantage ViewPoint** [citation:6].

**Step 2:** Enter your username and password [citation:6].

**Step 3:** Optionally, enable "automatically log on as this user" for convenience [citation:6].

### Step 4: Post-Installation Configuration

**Step 1:** Access the TeleVantage Administrator to configure system settings [citation:8].

**Step 2:** Set up call routing and IP telephony [citation:8].

**Step 3:** Configure users and extensions.
